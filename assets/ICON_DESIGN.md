# 图标设计说明

## 设计理念

新的图标设计体现了工作流（Workflow）和节点编辑器的核心理念：

- **节点网络**：四个角落的节点代表工作流中的不同步骤
- **中心节点**：中间的节点代表核心处理逻辑
- **连接线**：节点之间的连接线展示工作流的流程和关系
- **渐变色彩**：使用现代化的渐变色（紫蓝色系）体现科技感和专业性

## 图标文件说明

### `logo.svg`
- **用途**：主要logo，用于侧边栏、导航栏等位置
- **尺寸**：64x64px，支持任意缩放
- **特点**：彩色渐变设计，包含完整的节点网络和连接线

### `favicon.svg`
- **用途**：浏览器标签页图标
- **尺寸**：64x64px
- **特点**：简洁版本，白色节点在渐变背景上，适合小尺寸显示

### `logo-icon.svg`
- **用途**：小尺寸图标（如工具栏按钮）
- **尺寸**：32x32px
- **特点**：简化版本，保留核心设计元素

### `logo-monochrome.svg`
- **用途**：单色版本，适合需要适配不同主题的场景
- **尺寸**：64x64px
- **特点**：使用 `currentColor`，自动适配文本颜色

## 使用指南

### 在主应用中
```tsx
import logoUrl from "../assets/logo.svg";
<img src={logoUrl} height={32} width={32} alt="logo" />
```

### 作为 Favicon
```tsx
import faviconUrl from "../assets/favicon.svg";
<link rel="icon" href={faviconUrl} type="image/svg+xml" />
```

## 颜色方案

主渐变色：
- 起始色：`#6366f1` (Indigo-500)
- 中间色：`#8b5cf6` (Violet-500)
- 结束色：`#ec4899` (Pink-500)

这些颜色与 Tailwind CSS 的调色板一致，易于在应用中保持一致。

## 设计原则

1. **可识别性**：即使在很小尺寸下也能清晰识别
2. **现代感**：使用渐变和简洁的几何形状
3. **关联性**：图标直接关联工作流/节点的概念
4. **灵活性**：提供多个版本以适应不同使用场景
